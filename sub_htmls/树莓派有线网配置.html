树莓派有线网配置

日期： 2025-4-22

Raspberry Pi network 树莓派4B

1. 系统烧录相关

重要发现：

  • SD卡在PC上的挂载位置： /boot/firmware
  • user-data文件：系统首次启动的配置文件，烧录时会写入默认配置

2. 网络配置

配置文件位置： /etc/netplan/01-netcfg.yaml

基础有线网络配置示例： 

network:
  version: 2
  ethernets:
    eth0:
      dhcp4: no
      addresses: [59.70.61.236/24]
      routes:
        - to: default
          via: 59.70.61.254
      nameservers:
        addresses: [223.5.5.5, 223.6.6.6]

3. 配置步骤

  1. 编辑网络配置文件： 
    sudo nano /etc/netplan/01-netcfg.yaml
  2. 应用配置： 
    sudo netplan generate
sudo netplan apply
  3. 验证网络连接： 
    ip addr show eth0
ping 59.70.61.254

注意事项：

  • 配置文件中的缩进必须使用空格，不能使用制表符
  • 如果出现 "warning:root:cannot call open vSwitch" 警告，对于基本网络配置可以忽略
  • 确保IP地址、网关等信息符合你的网络环境
  • nameservers 使用了阿里DNS服务器地址（223.5.5.5, 223.6.6.6）

4. 可选配置

如果需要配置WiFi，可以在配置文件中添加 wifis 部分： 

  wifis:
    wlan0:
      optional: true
      dhcp4: true
      access-points:
        "WiFi名称":
          password: "WiFi密码"
